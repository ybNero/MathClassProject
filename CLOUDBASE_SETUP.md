# 腾讯云开发（CloudBase）接入指南

> 完成以下步骤后，学生提交的数据就会保存到云端，教师在任何设备都能看到。

---

## 第一步：创建云开发环境

1. 打开 [腾讯云开发控制台](https://console.cloud.tencent.com/tcb)
2. 点击「新建环境」→ 选择「按量付费」（有免费额度，够用）
3. 地域选 **上海**（国内延迟最低），记下**环境 ID**（格式类似 `xxx-abc123456`）

---

## 第二步：开启「未登录访问」

> 因为学生不需要微信账号，使用匿名访问即可。

1. 在云开发控制台左侧菜单 → **安全配置**
2. 找到「Web 安全域名」→ 添加 `mathclass01.netlify.app`
3. 找到「登录方式」→ 开启 **匿名登录**

---

## 第三步：创建数据库集合

1. 左侧菜单 → **数据库**
2. 新建集合，名称填：`tess_submissions`
3. 集合权限设置为：
   - **所有用户可读写**（学生提交、教师读取都需要）
   - 或按需使用自定义安全规则（见下方）

### 推荐安全规则（可选，更安全）

在集合「权限设置」→「自定义安全规则」中填入：

```json
{
  "read": true,
  "write": true
}
```

这允许任何匿名用户读写，适合教学场景。

---

## 第四步：修改 index.html 中的配置

打开 `index.html`，找到顶部的配置项（大约第 490 行）：

```js
const CB_ENV_ID = 'YOUR_ENV_ID';    // ← 替换为你的环境 ID
const CB_REGION = 'ap-shanghai';    // ← 如果创建时选的北京，改为 ap-beijing
```

将 `YOUR_ENV_ID` 替换为第一步记下的环境 ID，例如：

```js
const CB_ENV_ID = 'prod-abc123456';
const CB_REGION = 'ap-shanghai';
```

---

## 第五步：重新部署到 Netlify

保存 `index.html` 后，重新部署到 Netlify（推送或拖拽上传）。

---

## 验证是否成功

1. 用学生身份提交一份作业
2. 打开浏览器控制台（F12），看到 `[CloudBase] 初始化成功` 即正常
3. 切换到教师端，刷新后能看到刚才的提交

---

## 降级机制

代码已内置**降级保护**：
- 如果 CloudBase 初始化失败（网络问题、未配置环境 ID），会自动降级回 localStorage
- 控制台会打印 `[CloudBase] 初始化失败，降级使用本地存储`
